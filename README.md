## 1. Giới thiệu
Cài đặt thuật toán **Particle Swarm Optimization (PSO)** nhằm giải quyết bài toán tìm cực tiểu của hàm: f(x,y) = x^2 + y^2$.

Code bao gồm 1 bản gốc và 3 biến thể cải tiến để so sánh hiệu quả hội tụ.

## 2. Danh sách mã nguồn

* **`standard_pso.c`**: Thuật toán PSO cơ bản (Gốc).
* **`ldiw_pso.c`**: Biến thể LDIW 
* **`x-pso.c`**: Biến thể X-PSO 
* **`n-pso.c`**: Biến thể N-PSO
